# 编写规范（宏包）

本文档定义`.sty`宏包的文档编写规范，通用规范见 [PlanCommon.md](PlanCommon.md)。

## 文档结构

文档由一级标题`# 包名`开头，然后依次是三个二级标题。

```
## 引入的宏包
## 自定义命令
## 自定义环境
```

### 引入的宏包

每个被`\RequirePackage`加载的第三方宏包一个三级标题，用宏包表格（两列`宏包`、`说明`）列出。宏包若有预配置（`\PassOptionsToPackage`、`\ctikzset`、`\lstset`等），在其表格下方追加预配置表格（两列`选项`、`说明`）。

### 自定义命令

每个或每组自定义命令一个三级标题，用命令表格（两列`命令`、`说明`）列出。有参命令在命令列带参数，参数表格放在命令表格之后。

### 自定义环境

每个或每组自定义环境一个三级标题。环境先给出代码块签名（仅`...`占位，不含示例内容），然后是参数表格，最后给出示例代码块。示例内容应具有系列性和一致性。
